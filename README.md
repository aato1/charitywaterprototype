# charitywaterprototype

CleanTap is an idle game seeking to promote awareness of charity:water's mission for college students through an interactive game! Through the use of charity:water's branding, upgrades tied to real-life water initiatives they use to help communities, progression to keep users engaged over a longer period of time, and links to donate and recieve more information about the charity, I hope that users will both have fun while learning and hopefully supporting a great cause!

The game uses HTML, CSS, and JS to provide a responsive and interactive experience right in your web browser. Because of this, I hope to deploy this as an application to demo in the future!

I was able to practice both technical and conceptual skills during this project! I loved brainstorming coming up with the name, as well as how to integrate the brand/charity's work every step of the way, which is what inspired the idle game format to begin with! As for the code, the project helped me develop root styles to use throughout the game, practice features like local storage to track upgrades and drop counts across sessions, as well as integrating other cool game features like random events, difficulty, and unlocks/milestones.

I think the biggest struggle was getting the intial clock and storage to work, as well as some overall layout changes every step of the way. However, through a combination of Copilot and reading/debugging the code on my own, I was able to overcome any logic errors.
